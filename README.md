# Portafolio Web Express

Práctica guiada de desarrollo y versionamiento de un portafolio web profesional.

## Archivos incluidos

- `index.html`: estructura principal del portafolio.
- `styles.css`: estilos visuales básicos.
- `comandos-git.md`: comandos sugeridos para completar el versionamiento en Git y GitHub.

## Enlaces para entregar

Cuando subas el proyecto a GitHub, reemplaza los siguientes enlaces con los tuyos:

1. Sitio en vivo:
   `https://TU-USUARIO.github.io/portafolio-profesional/`

2. Historial de commits:
   `https://github.com/TU-USUARIO/portafolio-profesional/commits/main`

3. Repositorio de GitHub:
   `https://github.com/TU-USUARIO/portafolio-profesional`
